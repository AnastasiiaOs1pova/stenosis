# Структура проекта

## `stenosis-ai/`

- `data/` — датасеты для обучения и тестирования моделей.
- `ml/` — ноутбуки с разведочным анализом данных, классическими ML-моделями и нейронной сетью.
- `models/` — сохранённые модели.

## `stenosis-model-main-pinn-norm-q/`

Реализация расчётной модели стеноза, в которой дополнительный перепад давления на стенозе вычисляется через расход `q`.

Модель сначала решает одномерную задачу кровотока в сосудах: на каждом временном шаге рассчитываются площадь сечения сосуда `S`, скорость потока `U` и давление `P`. Расход через сечение определяется как:

```text
q = S · U
```

Для области стеноза используется специальное граничное условие `stenosis_pinn`, которое соединяет два сосуда. В этой точке модель:

1. вычисляет текущий расход через стеноз `q`;
2. по расходу находит среднюю скорость потока:

```text
V_ref = |q| / A_ref
```

где `A_ref = π · R_ref²`;

3. рассчитывает число Рейнольдса:

```text
Re = 2 · ρ · V_ref · R_ref / μ
```

4. передаёт в нейросетевую модель параметры:

```text
log(Re), Lr, Ds, asym
```

где `Lr` — относительная длина стеноза, `Ds` — степень стеноза, `asym` — параметр асимметрии;

5. нейросеть предсказывает нормированный перепад давления;
6. нормированный перепад переводится в размерный перепад давления в мм рт. ст.;
7. полученный дополнительный перепад добавляется в граничное условие между двумя сосудами.

## `stenosis-model-main-pinn-norm-u/`

Реализация расчётной модели стеноза, в которой дополнительный перепад давления на стенозе вычисляется через скорость потока `u`.

Модель решает одномерную задачу кровотока в сосудистой сети. На каждом временном шаге рассчитываются площадь поперечного сечения сосуда `S`, скорость потока `U` и давление `P`. Для области стеноза используется граничное условие `stenosis_pinn`, которое соединяет два сосуда и добавляет дополнительный перепад давления, предсказанный нейросетевой моделью.

В этой версии расчёт строится не через расход `q`, а через скорость на входе в стеноз:

```text
u_pinn = ORI[0] · u_end[0]
```

где `u_end` — скорость на граничном узле сосуда, а `ORI[0]` учитывает направление потока.

Далее по текущей площади сечения вычисляется динамический радиус:

```text
R_dyn = sqrt(area0 / π)
```

Относительная длина стеноза пересчитывается с учётом этого радиуса:

```text
Lr_dyn = L_st_phys / R_dyn
```

где:

```text
L_st_phys = Lr · R_ref
```

Затем рассчитывается сглаженный модуль скорости:

```text
u_abs_smooth = sqrt(u_pinn² + u_eps²)
```

и по нему находится число Рейнольдса:

```text
Re = 2 · ρ · u_abs_smooth · R_dyn / μ
```

После этого в нейросетевую модель передаются параметры:

```text
Re, Lr_dyn, Ds, ρ, μ, R_dyn, asym
```

где `Ds` — степень стеноза, `asym` — параметр асимметрии.

Нейросеть предсказывает дополнительный перепад давления на стенозе `dp_extra_mmhg`. Этот перепад добавляется в условие баланса давления между двумя сосудами:

```text
P_RHO[0] - P_RHO[1] - dp_extra_mmhg = 0
```

При этом также выполняется условие сохранения расхода:

```text
Σ ORI · area · u = 0
```
